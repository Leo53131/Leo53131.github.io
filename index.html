HTML

HTML son las siglas de HyperText Markup Language. es el lenguaje de marcado principal para la cracion de paginas web

Creado por Tim Bernaers-lee, HTML ha evolucionado significativamente dede sus inicios

Historia de HTML

Los inicios del lenguaje HTML se remontan a principios de la decada de 1990. Desde entonces, ha pasado por varias versiones, cada una añadiendo nuevas funciones y mejoras.

Versiones de HTML

Tim Bernaers-lee definio la primera version de HTML en el año 1991.

En la actualidad, la ultima version de HTML es HATM5
